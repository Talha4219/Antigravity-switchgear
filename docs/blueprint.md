# **App Name**: SwitchGear Pro

## Core Features:

- Product Catalog: A detailed catalog displaying switchgear products with specifications, applications, and high-quality images.
- Dynamic Blog Management: Admin dashboard to manage and publish blog posts, industry insights, and company updates.
- Offer Management: Admin interface to create and manage latest offers and promotions on switchgear products.
- Certifications Showcase: Dedicated page displaying company certifications, compliance standards, and legal credentials to build trust.
- Interactive Contact Form: A comprehensive 'Contact Us' page with an interactive form, location details, and prompt communication options.
- Expertise Elucidation: An 'About Us' section utilizing generative AI to elaborate on the company's expertise, history, and core values in an engaging manner. An AI tool will assist in elaborating expertise related items to improve site presentation.

## Style Guidelines:

- Primary color: Navy blue (#192A56) for a professional and reliable feel.
- Background color: Light gray (#F5F7FA) for a clean and modern aesthetic.
- Accent color: Silver (#A0AEC0) for highlights and accents to add a touch of sophistication.
- Body font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look and clear readability.
- Headline font: 'Space Grotesk' (sans-serif) to be paired with Inter, adding a computerized, techy, scientific feel, making it stand out from the body.
- Use professional and consistent icons relevant to switchgear products and industrial applications.
- Clean and structured layout with clear sections for easy navigation and information access.
